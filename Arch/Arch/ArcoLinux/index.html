# ArcoLinux

super+shfit+a，程序启动窗口

super+Q，退出

super + 鼠标单击，可以拖动窗口

super+Enter，打开默认 urxvt 终端模拟器，附带三个，termite,xfec 自带终端和 alacritt

Super+Space，修改当前窗口布局

软件包管理工具有 pacman,pamac,yay
