# 功能键

WinAltCltrShiftAlt Gr
#!^+<^>!

其中， <, > 为修饰符，用于区分成对出现的按键，例如 <! 表示左侧的 Ctrl 。按照这个逻辑理解， <^>! 应该表示左 Ctrl + 右 Alt ，对于某些具有 Alt Gr 键的电脑而言，则专门指代这个按键。
